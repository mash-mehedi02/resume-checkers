# Quick Run Guide

## Fastest Way to Run (VS Code)

1. **Install Java Extension Pack:**
   - VS Code → Extensions (Ctrl+Shift+X)
   - Search: "Java Extension Pack"
   - Install

2. **Run Application:**
   - Open `ResumeScreenerApplication.java`
   - Click "Run" button (▶️) or press F5
   - VS Code will automatically build and run

## Alternative: Use run.bat

Double-click `run.bat` file (if Maven is installed)
